# TaskTracker

A simple .NET console app for managing tasks. Used as a test project for Maestro dev workflow testing.

## Build & Run

```bash
dotnet build
dotnet run
```

## Commands

```
add <title>          — Add a new task
list                 — List all tasks
complete <id>        — Mark a task complete
remove <id>          — Remove a task
stats                — Show task statistics
search <keyword>     — Search tasks by title
export               — Export tasks to CSV
quit                 — Exit
```
