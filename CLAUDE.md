# TaskTracker

A simple .NET 9 console application for managing tasks.

## Structure

- `Program.cs` — CLI entry point (REPL loop)
- `Models/TaskItem.cs` — Task data model
- `Services/TaskService.cs` — Task operations (add, remove, complete, search, stats, export)

## Build & Run

```bash
dotnet build
dotnet run
```
